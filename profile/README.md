# Edgefall Studios

**Independent module studio for Foundry VTT.**

I build modules for Foundry VTT — designed for V13 and V14, with
dnd5e (2024) as the primary system and Pathfinder 2E supported
where the design fits. Modules ship when they're solid, not before.

Repos here are public **support repos** for shipped modules
(install instructions, issue tracker, releases, changelog). Source
for premium modules stays private and is delivered through Foundry's
Patreon integration.
